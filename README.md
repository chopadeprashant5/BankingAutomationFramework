# Enterprise Hybrid Automation Framework – Online Banking System

## 📌 Project Overview
This project is a Selenium Hybrid Automation Framework designed to automate
critical banking workflows such as login, account validation, and transaction testing.

## 🛠 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Jenkins (CI/CD)
- GitHub

## 📊 Features
✔ Page Object Model  
✔ Data Driven Testing  
✔ Extent HTML Reports  
✔ Jenkins Continuous Integration  

## 🚀 Execution
Run tests using:

mvn clean test

## 📈 CI Pipeline
GitHub → Jenkins → Maven → Test Execution → Extent Report
